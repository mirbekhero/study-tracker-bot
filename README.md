Study Reminder Bot

Умный Telegram-бот для студентов, который помогает не забывать о задачах и превращает учебу в игру.

 Функции
* **Список задач:** Добавляй дела простым сообщением.
* **Геймификация:** Получай XP (опыт) за каждую выполненную задачу.
* **Ранги:** Пройди путь от "Амебы" до "Магистра Дедлайнов".
* **База данных:** SQLite сохранит твои задачи даже после перезагрузки.

Технологии
* Python 3.14
* Aiogram 3 (библиотека для бота)
* SQLite3 (база данных)
* Python-dotenv (конфигурация)

 Как запустить
1. Склонируй репозиторий.
2. Создай файл `.env` и добавь туда свой `BOT_TOKEN`.
3. Установи зависимости:
   ```bash
   pip install aiogram python-dotenv
